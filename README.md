# 📮 Hulak

> /hu-laak/ — Post Office

Hulak is a simple web app for sending thoughtful digital letters and postcards.

Write something meaningful, or completely unhinged, wrap it in a digital envelope, and send it to someone special.

It’s an experiment in slowing down digital communication and making it feel a bit more intentional again.

---

## ✨ What it does

- Create and send digital letters  
- Wrap messages in styled envelopes  
- Attach stamp themes and visual styles  
- Generate shareable letter links  
- Open and read letters in a dedicated view  

Each letter becomes its own small “postcard” on the internet, accessible through a unique URL.

## 🧠 Why I built this

Hulak started as a personal project to explore **[Astro.js](https://astro.build/)** and build something beyond tutorials and toy examples.

Along the way it also became:

- A playground for UI and animation experiments  
- A place to explore digital postcard and stamp aesthetics  
- A side project built around semester chaos at **[Kathmandu University](https://ku.edu.np/)**  

It is not trying to reinvent communication. Just slow it down a bit.

## 💭 Philosophy (kind of)

Hulak is built around a simple idea:

Not every message needs to be instant.

Some things can sit in a small envelope, take a second to open, and feel slightly more intentional than a chat notification.
